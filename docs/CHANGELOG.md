# 变更日志

本文档记录项目的主要功能变更和问题修复。

## 版本历史

### v1.1.0+ (2026-02 ~ 2026-04)

#### 新功能

- **夜间模式主题系统**
  - 日间/夜间双主题，Ctrl+T 快速切换
  - ThemeManager 单例管理主题颜色
  - 40+ 颜色字段覆盖所有 UI 元素
  - 自定义 SpinBox 三角形箭头（替代 Qt 默认按钮）

- **图片处理增强**
  - 程序展示图片：等比缩放不裁剪（偏高锁高度、偏宽锁宽度）
  - 模型/车辆/精彩瞬间：居中裁剪为 4:5
  - 所有图片类型最大张数统一为 9 张
  - 图片上传独立同步按钮（每类图片可独立控制是否同步）

- **学员独立课时编号**
  - 每个学员独立的课时进度跟踪
  - 生成 PPT 后自动 +1
  - 替代旧的班级级课时编号

- **配置导出/导入系统**
  - 全量导出/导入（v1.1.0 基础版）
  - 选择性导出/导入（v1.1.0+ 增强）
    - 按班级选择导出范围
    - 可选包含课程系列、颜色配置、最近记录、默认值
    - 导入时同名班级冲突处理：创建副本或跳过
    - 学员合并去重、课时编号取较大值
  - 夜间模式对话框样式适配

- **课堂知识编号自动清理**

- **PPT 保存后自动另存为 PDF**

- **启动检测**
  - 字体依赖检测（华文琥珀）
  - PowerPoint 安装检测

- **其他**
  - 班级记忆 PPT 保存路径
  - 班级母版配置联动
  - 生成 PPT 前内容检查提示
  - 将"作品展示"更名为"精彩瞬间"
  - 应用更名为"法贝实验室课程反馈助手"

#### Bug 修复

| 问题 | 解决方案 |
|------|----------|
| 夜间模式下清除颜色后文字变黑 | 清除时使用主题 text_primary 色而非固定黑色 |
| 车辆展示页面不生成 | 修复布局索引映射 |
| 补充说明同步后切换学员仍触发未修改警告 | 修正同步状态重置时机 |
| 小名是大名后缀时独立小名未被替换 | 改进替换逻辑，先替换小名再替换大名 |
| 补充说明同步功能缓存数据不完整 | 补充缺失字段 |
| 夜间模式对话框白底灰字看不清 | 对话框自行应用主题 stylesheet |
| PDF 导出 Intent 参数错误 | 修正参数传递 |
| 多图片上传时幻灯片顺序错误 | 修正排序逻辑 |
| 批量生成时未点击标签的学生评价不正确 | 修复评价数据获取逻辑 |
| 第2课 PPT 生成问题 | 修复课时编号计算 |
| QByteArray.hex() 方法不存在 | 改用 bytes.hex() |
| QFontDatabase 依赖问题 | 移除 QFontDatabase 使用替代方案 |
| QToolBar saveState 警告 | 设置 QToolBar objectName |

#### 重构

- 删除未使用的 PPT 预览功能
- 精简 PPT 模板文件
- 合并 PPT 后处理为单次打开（性能优化）
- 取消精彩瞬间图片同步，改为学员独立

## 已知问题

1. **Excel 导入**：仅支持特定格式的 Excel 文件
2. **图片裁剪**：不支持旋转后的裁剪区域调整
3. **批量生成**：大量生成时内存占用较高
4. **跨平台**：目前仅支持 Windows

## 待优化方向

1. **架构重构**：main_window.py 过大（~2800行），应拆分
2. **单元测试**：添加 pytest 测试用例
3. **自定义模板**：支持多种 PPT 模板
4. **数据备份**：自动备份配置文件
