## 二、手动测试流程

### 1. 启动程序
```bash
python main.py
```

### 2. 基础界面测试
- [ ] 程序正常启动，显示主窗口
- [ ] 窗口标题为"课程反馈PPT自动生成器"
- [ ] 菜单栏包含：文件、编辑、视图、帮助
- [ ] 工具栏显示：新建、打开、保存、生成PPT
- [ ] 状态栏显示"就绪"
- [ ] 关闭窗口后再次打开，窗口位置/大小被记住

### 3. 基本信息Tab测试
- [ ] **课程信息输入**
  - 课时编号可调整
  - 课程内容可输入
  - 学生姓名可输入（有历史记录补全）
  - 授课教师可输入（有历史记录补全）
  - 课时数可调整
  - 日期时间可选择

- [ ] **母版选择**
  - 各复选框可勾选/取消
  - 模型展示页数量可调整（1-10）
  - "恢复默认"按钮正常工作

### 4. 课堂评价Tab测试
- [ ] **12项评价**
  - 每项评价有5个选项：优/良/中/差/未体现
  - 每项只能选择一个
  - 选择后其他选项自动取消

- [ ] **总体评价**
  - 有4个选项：优/良/仍需努力/需要改进

- [ ] **上次作业情况**
  - 有5个选项：优/良/中/差/无

- [ ] **补充说明/课堂作业/注意事项**
  - 文本框可输入

### 5. 图片上传Tab测试
- [ ] **模型展示图片**
  - 点击"添加图片"可选择图片
  - 拖拽图片到区域可上传
  - 图片显示预览
  - 鼠标悬停显示删除按钮
  - 点击删除按钮可删除图片
  - "清空"按钮清除所有图片

- [ ] **作品展示图片**
  - 同上功能

### 6. PPT生成测试
- [ ] 输入必填信息（学生姓名、课程内容）

- [ ] PPT成功生成

  - 评价复选框位置正确（不换行）

## 四、预期输出

### 生成的PPT结构
1. 第1页：封面页
2. 第2页：课程信息页（包含评价、作业等）
3. 第3页：模型展示页（带图片）
4. 第4页：作品展示页（带图片）

### 文件命名
格式：`学生姓名_第X课_课程反馈_日期时间.pptx`

## 五、常见问题

### Q1: PPT中复选框位置错乱
A: 已修复，使用模板原有Wingdings字符

### Q2: 图片无法上传
A: 检查图片格式（支持jpg/png/bmp/gif/webp）

### Q3: Excel导入失败
A: 确保使用正确的模板格式，第一行为表头

### Q4: 中文显示乱码
A: 确保系统编码为UTF-8
