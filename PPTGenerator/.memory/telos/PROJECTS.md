# 项目 (Projects)

## 当前活跃项目

### 课程反馈 PPT 自动生成器

**项目类型**: 桌面应用 (PyQt5 + python-pptx)

**项目描述**:
自动化生成课程反馈 PPT，支持：
- 多种母版布局选择
- 课程信息填充
- 重点/难点文本标记
- 评价选项处理
- 图片嵌入
- 批量生成

**技术栈**:
- Python 3.x
- PyQt5 (GUI)
- python-pptx (PPT 操作)
- Pillow (图片处理)
- openpyxl (Excel 导入)

**项目结构**:
```
D:/claude code/
├── main.py                    # 主入口
├── src/
│   ├── core/                  # 核心功能模块
│   └── ui/                    # 界面组件
├── templates/                 # PPT 模板
├── config/                    # 配置文件
├── .agent/                    # 增量开发系统
└── .memory/                   # 记忆系统
```

**当前版本**: 1.0.0

**完成状态**: 核心功能 100% 完成
