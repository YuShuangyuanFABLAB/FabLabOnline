# 挑战 (Challenges)

## 已解决的挑战

### C001: PPT 颜色验证困难
**问题**: python-pptx 读取的颜色与实际显示不一致
**解决方案**:
- 使用 PowerPoint COM API 进行验证
- 创建 skill 文档记录方法
- 建立三层验证体系

**学到的**:
- 不能只相信代码层的测试
- 必须用目标环境的原生 API 验证

### C002: 会话上下文丢失
**问题**: 每次新会话 AI 都要从头了解项目
**解决方案**:
- 建立 .agent 增量开发系统
- 使用 feature_list.json 追踪功能
- 使用 agent-progress.txt 记录进度

**学到的**:
- 结构化的进度文件是跨会话的关键
- 要有标准化的会话开始流程

### C003: 组合形状内文本处理
**问题**: PPT 中的组合形状内文本无法直接访问
**解决方案**:
- 实现递归查找算法
- 遍历所有子形状

**学到的**:
- PPT 结构比想象的更复杂
- 需要处理各种边缘情况
