# 法贝实验室课程反馈助手

> 自动生成课程反馈 PPT 的桌面应用程序，适用于教育培训机构快速生成标准化、个性化的课程反馈报告。

## 功能特性

### 班级与学员管理
- **多班级支持**：创建和管理多个班级，每个班级可设置独立的上课时间和默认教师
- **班级记忆路径**：每个班级记住上次 PPT 保存路径
- **班级母版联动**：班级关联课程系列配置
- **学员管理**：添加/编辑/删除学员，支持昵称
- **学员标签栏**：底部快速切换学员，每位学员数据独立缓存
- **独立课时编号**：每个学员独立的课时进度跟踪，生成 PPT 后自动 +1

### 课程信息录入
- **课程系列选择**：预设多种课程系列（机械臂设计、玩具大改造、军卡大改造等）
- **自动填充**：课程内容、教师姓名等根据班级配置自动填充
- **历史记录**：学生/教师姓名自动补全
- **课堂知识编辑**：富文本编辑器，支持重点（蓝色）/难点（橙色）标记
- **编号自动清理**：课堂知识编号自动清理功能

### 课堂评价系统
- **12 项专业评价**：专注度与效率、课堂听课习惯、任务完成、内容理解等
- **评价等级**：优、良、中、差、未体现
- **总体评价**：优、良、仍需努力、需要改进
- **作业评价**：上次作业情况记录
- **补充说明**：支持智能同步，可按需独立编辑

### 图片管理
- **拖拽/点击上传**：支持拖拽或点击选择图片
- **图片预览与裁剪**：缩略图预览，内置裁剪功能
- **多种图片类型**：模型展示、程序展示、车辆展示、精彩瞬间
- **智能处理策略**：
  - 程序展示图片：等比缩放不裁剪（偏高锁高度、偏宽锁宽度）
  - 模型/车辆/精彩瞬间：居中裁剪为 4:5
- **最多 9 张**：所有图片类型最大张数统一为 9 张
- **独立同步**：每类图片可独立控制是否同步

### PPT 生成
- **模板驱动**：基于预设模板自动生成 PPT
- **布局配置**：可选择包含课程信息页、模型展示、程序展示、车辆展示、精彩瞬间等模块
- **批量生成**：支持一次为多个学员批量生成 PPT
- **生成前检查**：生成前自动检查内容完整性
- **自动导出 PDF**：PPT 保存后自动另存为 PDF

### 主题系统
- **日间/夜间双主题**：`Ctrl+T` 快速切换
- **40+ 颜色字段**：全面覆盖所有 UI 元素
- **自定义 SpinBox**：三角形箭头替代 Qt 默认按钮
- **自动保存**：主题设置自动持久化

### 配置导出/导入
- **选择性导出**：按班级选择导出范围，可选包含课程系列、颜色配置等
- **选择性导入**：预览导入内容，按需选择班级
- **冲突处理**：同名班级可选择创建副本或跳过
- **智能合并**：学员去重、课时编号取较大值、评价记录覆盖
- **便携版迁移**：方便在不同电脑间迁移配置

### 启动检测
- **字体依赖检测**：自动检测华文琥珀等必需字体
- **PowerPoint 检测**：检测 PowerPoint 安装状态

## 安装与运行

### 环境要求
- Python 3.8+
- Windows

### 安装依赖

```bash
pip install PyQt5 python-pptx Pillow lxml
```

### 运行程序

```bash
python main.py
```
