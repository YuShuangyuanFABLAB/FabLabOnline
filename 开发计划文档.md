# 课程反馈PPT自动生成程序 - 开发计划文档

## 一、项目概述

### 1.1 项目目标
开发一个Windows桌面GUI程序，基于已有的PPT模板，自动为学生家长生成课程反馈PPT。程序支持手动输入、Excel批量导入、表单填写等多种数据输入方式，自动处理文本格式、评价选项、图片裁剪等功能。

### 1.2 技术栈
| 类别 | 技术选型 | 说明 |
|------|----------|------|
| 编程语言 | Python 3.8+ | 跨平台兼容性好 |
| PPT操作 | python-pptx | 成熟的PPT处理库 |
| GUI框架 | PyQt5 | 功能强大，界面美观 |
| 图片处理 | Pillow (PIL) | 图片裁剪、缩放 |
| Excel处理 | openpyxl | 读写Excel文件 |
| 配置存储 | JSON | 保存用户偏好设置 |

## 二、功能需求详细说明

### 2.1 母版布局管理

#### 2.1.1 支持的母版类型
| 布局编号 | 布局名称 | 用途说明 | 页面位置 |
|----------|----------|----------|----------|
| 0 | 4_标题和内容 | 首页（仅模板使用，不新增） | - |
| 1 | 标题和内容 | 课程信息页 | 每单元第1页 |
| 2 | 3_标题和内容 | 精彩瞬间/作品页 | 每单元最后页 |
| 3 | 5_标题和内容 | 车辆展示页 | 可选 |
| 4 | 2_标题和内容 | 程序展示页 | 可选 |
| 5 | 1_标题和内容 | 精彩瞬间页(双图) | 可选 |
| 6 | 6_标题和内容 | 模型展示页 | 每单元中间页 |

#### 2.1.2 母版选择逻辑
```
默认单元结构（用户可自定义）：
├── 课程信息页（必选）- 标题和内容
├── 模型展示页（可选，可多页）- 6_标题和内容
├── 精彩瞬间双图页（可选）- 1_标题和内容
├── 程序展示页（可选）- 2_标题和内容
├── 车辆展示页（可选）- 5_标题和内容
└── 作品页（必选）- 3_标题和内容
```

#### 2.1.3 母版记忆功能
- 首次使用：用户手动勾选需要的母版组合
- 后续使用：自动加载上次的选择（从config/settings.json读取）
- 提供"恢复默认"按钮重置选择

### 2.2 课程信息页输入项

#### 2.2.1 基本信息区域
| 字段名 | 数据类型 | 是否必填 | 说明 |
|--------|----------|----------|------|
| 课时编号 | 整数 | 是 | 如：1、2、3... |
| 课程内容 | 文本 | 是 | 如：机械臂设计（1阶） |
| 学生姓名 | 文本 | 是 | 可保存常用学生列表 |
| 授课教师 | 文本 | 是 | 可保存常用教师列表 |
| 课时数 | 整数 | 是 | 如：2 |
| 上课日期 | 日期时间 | 是 | 格式：2025.7.23 18:30-20:30 |

#### 2.2.2 课堂知识内容区域
| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| 课程内容文本 | 多行文本 | 支持富文本格式 |
| 重点内容标记 | 标记功能 | 选中文字标记为蓝色 #0070C0 |
| 难点内容标记 | 标记功能 | 选中文字标记为橙色 #ED7D31 |

**文本编辑器功能**：
- 支持选中文本后点击按钮标记为重点/难点
- 支持取消标记
- 实时预览效果
- 支持多段落输入

### 2.3 图片页处理

#### 2.3.1 图片上传功能
- 支持 drag & drop 拖拽上传
- 支持点击选择文件
- 支持批量选择图片
- 支持的格式：JPG、PNG、BMP、GIF

#### 2.3.2 图片处理流程
```
原图上传 → 预览显示 → 裁剪调整 → 嵌入PPT
```

#### 2.3.3 图片裁剪参数
| 母版类型 | 目标尺寸 | 裁剪方式 |

| 3_标题和内容 | 6.55in x 8.07in | 居中裁剪/手动裁剪 |

#### 2.3.4 图片处理功能
- 自动缩放：保持宽高比，适配目标区域

- 预览功能：显示裁剪后效果

### 6.2 数据模型

```python
from dataclasses import dataclass
from typing import Optional, List
from enum import Enum
from datetime import datetime

class EvaluationLevel(Enum):
    """评价等级"""
    EXCELLENT = "优"
    GOOD = "良"
    MEDIUM = "中"
    POOR = "差"
    NOT_SHOWN = "未体现"

class OverallEvaluation(Enum):
    """总体评价"""
    EXCELLENT = "优"
    GOOD = "良"
    NEED_EFFORT = "仍需努力"
    NEED_IMROVEMENT = "需要改进"

class HomeworkEvaluation(Enum):
    """作业评价"""
    EXCELLENT = "优"
    GOOD = "良"
    MEDIUM = "中"
    POOR = "差"
    NONE = "无"

@dataclass
class CourseUnitData:
    """课程单元数据"""
    # 基本信息
    lesson_number: int
    course_content: str
    student_name: str
    teacher_name: str
    class_hours: int
    class_date: str

    # 课程内容
    knowledge_content: str
    highlights: List[str]  # 重点内容列表
    difficulties: List[str]  # 难点内容列表

    # 课堂评价
    logic_thinking: EvaluationLevel
    content_understanding: EvaluationLevel
    task_completion: EvaluationLevel
    listening_habit: EvaluationLevel
    problem_solving: EvaluationLevel
    independent_analysis: EvaluationLevel
    knowledge_proficiency: EvaluationLevel
    imagination_creativity: EvaluationLevel
    frustration_handling: EvaluationLevel
    learning_method: EvaluationLevel
    hands_on_ability: EvaluationLevel
    focus_efficiency: EvaluationLevel

    # 总体评价
    overall_evaluation: OverallEvaluation

    # 补充信息
    additional_comments: str
    homework: str
    other_notes: str
    last_homework_status: HomeworkEvaluation

    # 图片路径
    model_images: List[str]
    work_images: List[str]
    double_images: List[str]  # 双图页

@dataclass
class LayoutConfig:
    """母版配置"""

    model_display_count: int = 1

    include_program_display: bool = False
```
